# Cultural Considerations Dataset for Malayalam AI IVR

## Overview

This dataset provides comprehensive cultural awareness patterns for both Malayalam and English interactions in the Kerala cultural context. The AI agent uses this dataset to ensure culturally appropriate and respectful communication.

## Cultural Intelligence Implementation

### 1. Politeness Level Detection

#### High Politeness Markers

- Malayalam: താങ്കൾ, അവിടുന്ന്, തിരുമുമ്പിൽ, അങ്ങ്, ബഹുമാനപൂർവം
- Manglish: thankall, avidunnu, thirumumshil, angu, bahumanapoorvam
- English: respected, honored, esteemed, graciously, humbly

#### Medium Politeness Markers

- Malayalam: നിങ്ങൾ, തങ്ങൾ, സാർ, മാഡം
- Manglish: ningall, thankall, sir, madam
- English: please, kindly, could you, would you

#### Low Politeness Markers

- Malayalam: നീ, ഇവൻ, ഇവൾ, അവൻ, അവൾ
- Manglish: nee, ivan, ival, avan, aval
- English: hey, what's up, buddy

### 2. Cultural Context Scoring

#### High Cultural Sensitivity (Score: 0.8-1.0)

- Religious references present
- Festival context detected
- Family hierarchy markers found
- Regional cultural indicators present

#### Medium Cultural Sensitivity (Score: 0.5-0.8)

- Respectful address patterns
- Age-appropriate language
- Gender-sensitive terms

#### Basic Cultural Awareness (Score: 0.3-0.5)

- Standard politeness markers
- Basic cultural terms
- Regional references

### 3. Response Generation Guidelines

#### For High Cultural Context

- Use most respectful forms of address
- Include cultural/religious blessings when appropriate
- Acknowledge festival contexts
- Show awareness of family hierarchy

#### For Medium Cultural Context

- Use polite and respectful language
- Include appropriate cultural markers
- Show sensitivity to age and gender

#### For Basic Context

- Use standard polite forms
- Maintain cultural neutrality while being respectful
- Use universally appropriate language

## Usage Examples

### Scenario 1: Festival Greeting

**Input**: "Onam aashamsakal! Engane irikkunnu?"
**Cultural Analysis**: Festival context (Onam) + informal greeting
**Response**: "ഓണാശംസകൾ! സുഖമായിരിക്കുന്നു. താങ്കൾക്കും കുടുംബത്തിനും ഓണത്തിന്റെ ആശംസകൾ!"

### Scenario 2: Respectful Elder Address

**Input**: "Namaskaram sir, oru sahayam cheyyamo?"
**Cultural Analysis**: High politeness + respectful address
**Response**: "ആദരപൂർവം നമസ്കാരം. താങ്കളുടെ സേവനത്തിൽ ഞാൻ. എന്ത് സഹായം വേണം?"

### Scenario 3: Family Context

**Input**: "Achan hospital il aanu, urgent aayitt oru auto venam"
**Cultural Analysis**: Family emergency + respect for elder
**Response**: "അച്ഛനു വേണ്ടി ഉടൻ തന്നെ വാഹനം ക്രമീകരിക്കാം. ആശുപത്രിയിൽ എത്തിക്കാൻ സഹായിക്കാം."

## Implementation Notes

1. **Priority Order**: Religious context > Family hierarchy > Age respect > General politeness
2. **Regional Adaptation**: Detect regional markers and adjust dialect accordingly
3. **Festival Awareness**: Maintain calendar awareness for appropriate seasonal greetings
4. **Gender Sensitivity**: Use appropriate pronouns and respectful terms based on cultural context
5. **Emergency Override**: In emergency situations, prioritize help while maintaining cultural respect
