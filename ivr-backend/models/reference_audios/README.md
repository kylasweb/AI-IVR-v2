# Reference Audio Files for IndicF5 Voice Cloning

This directory stores reference audio files for zero-shot voice cloning with IndicF5.

## Requirements

Each audio file should be:
- **Duration**: 5-15 seconds
- **Format**: WAV (16-bit PCM)
- **Sample Rate**: 24000 Hz (preferred) or 16000 Hz
- **Content**: Clear Malayalam speech by a native speaker

## File Naming

Name files by district (lowercase, no spaces):
```
thiruvananthapuram.wav
kollam.wav
pathanamthitta.wav
alappuzha.wav
kottayam.wav
idukki.wav
ernakulam.wav
thrissur.wav
palakkad.wav
malappuram.wav
kozhikode.wav
wayanad.wav
kannur.wav
kasaragod.wav
```

## Recording Tips

1. Use a quiet environment with minimal background noise
2. Speak naturally in the regional dialect
3. Include common phrases or greetings
4. Avoid music or sound effects

## Optional

Without reference audio, IndicF5 will use its default Malayalam voice (still high quality, but without dialect-specific characteristics).
